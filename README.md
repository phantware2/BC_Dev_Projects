## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/phantware2/BC_Dev_Projects.git
```

2. **Navigate to the project directory:**

```bash
cd BC_Dev_Projects
```

3. **Follow individual project instructions** in their respective folders.
